# Installation<a id="orgheadline2"></a>

-   Clone the project
-   `cd Abstraction-LeapMotionServer`
-   `make`

# Usage<a id="orgheadline3"></a>

-   Connect your Leap Motion and run `sudo leapd`
-   `./bin/LeapServer <nb_client> <port>`
